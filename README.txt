Курсовой проект

Автор: Алексей Илларионов


В качесте модели для курсового проекта я выбрал систему доставки готовой еды "Яндекс.Еда".
В данной системе БД используется для хранения информации о блюдах ресторанов, информации о самих ресторанах, пользователях системы, а также для хранения финансовой информации.

Основными таблицами для хранения данных в этом проекте являются таблицы:

users - используется для хранения информации о пользователях;
users_addresses - адреса пользователя;
companies - хранит информацию о юрлицах;
restorans - содержит информацию о ресторанах;
menu_categories - категории меню;
menu_items - позиции меню;
orders - информация о заказах;
delivery_zone - информация о зоне доставки;
sale - скидки;
